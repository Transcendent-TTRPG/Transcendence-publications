# Transcendence Technique Cards

Prototype printable tarot-size cards for table play.

- Format: US Letter sheet with four tarot cards per page.
- Card size: 70 mm x 120 mm.
- Source examples: real entries from `Transcendence-design/data/system/techniques.yaml`.
- Current status: static layout prototype. Later this can be generated from YAML.

Open `index.html` in a browser and print at 100% scale.
