# 04-skills-and-proficiencies

Language folders:

- es/ source in Spanish
- en/ mirror translation in English

Recommended internal structure:

1. `specializations.md` / `especializaciones.md`
   Core explanation of what specializations are, how they work, categories, starting access,
   progression, Synapsis, and relation to Techniques.
2. `specializations-catalog.md` / `catalogo-de-especializaciones.md`
   Reader-facing compendium of individual specializations.

Editorial note:

- The detailed specialization compendium belongs in this chapter, not in Character Creation,
  Techniques, or Equipment.
- `Character Creation` should reference available starting options, but not carry the full catalog.
- `Techniques` should consume specialization prerequisites, not redefine the specializations themselves.
- If the catalog becomes too large, split it later by category inside the same chapter rather than moving it to another chapter.
