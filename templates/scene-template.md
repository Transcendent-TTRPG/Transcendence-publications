# Scene Template

Use this template to draft a narrative scene or prose fragment.

## Purpose

Use for fiction scenes, atmospheric inserts, in-world narrative fragments, or short prose passages.

Before writing, read:

1. `canon/text-modes.md`
2. `canon/style-guide.md`
3. `canon/voice-samples.md`
4. `canon/glossary.md`
5. `canon/fiction-guide.md`

## Mode Declaration

- Primary mode: `narrative` or `description`
- Secondary mode if needed: `flavor`
- If the passage is actually explaining a system, stop and switch to `rules` or `example`

## Scene Function

State what this scene is doing.

Choose one:

- reveal pressure
- establish place
- present anomaly
- show consequence
- transition between larger scenes
- reveal character through action

## Scene Frame

- Viewpoint:
- Time:
- Location:
- Active source of tension:
- Is the tension ordinary, hostile, or extranatural?

## Narrative Guidance

- One active perceptual thread per paragraph
- Show change over time, not static summary
- Do not explain hidden causes directly
- Use one primary sensory channel per paragraph
- If Limbo / horror-facing, the anomaly must be physically perceivable
- If ordinary fiction, the paragraph still needs tension, pressure, change, or consequence

## Draft Structure

```md
# [Scene Title]

## Scene Purpose

[One or two lines on what this scene must accomplish.]

## Passage

[Paragraph 1: base state -> deviation]

[Paragraph 2: consequence -> response]

[Paragraph 3: escalation, restraint, or unresolved carry-forward]
```

## Do Not Do

- do not explain mechanics inside the scene
- do not stack metaphors
- do not label emotion without physical grounding
- do not use scene text to dump lore
