# Supplement Template

Use this template to start a supplement or short publication unit.

## Purpose

Use for a small publication unit: species guide, faction guide, rules add-on, setting dossier, short adventure packet, or similar.

Before writing, read:

1. `canon/text-modes.md`
2. `canon/style-guide.md`
3. `canon/glossary.md`
4. Relevant canon and design sources for the supplement topic

## Supplement Type

Choose one before drafting:

- mechanical supplement
- setting supplement
- hybrid supplement
- adventure supplement

## Mode Profile

### Mechanical supplement

- primary modes: `rules`, `reference`
- secondary mode: `example`

### Setting supplement

- primary modes: `description`
- secondary modes: `flavor`, controlled `narrative`

### Hybrid supplement

- split by section; do not blend modes casually

### Adventure supplement

- primary modes: `description`, `reference`
- secondary modes: `narrative`, `example`

## Reader Contract

State clearly:

- what the supplement helps the reader do
- what kind of information is authoritative here
- what information is only illustrative

## Recommended Outline

```md
# [Supplement Title]

## What This Supplement Covers

[Scope and reader need.]

## Required Context

[Canon, rules, setting, or continuity assumptions.]

## Main Sections

### [Section]

- Primary mode:
- Reader task:
- Must include example? yes/no
```

## Constraints

- If a section defines mechanics, treat it as `rules`
- If a section only orients the reader to a place, faction, or phenomenon, treat it as `description`
- Keep examples separate from definitions
- If the supplement becomes large enough to need multiple mode families, split it into subsections deliberately
