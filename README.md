# Transcendence Publications

This repository contains the editorial and publication-facing content for the Transcendence project.

## Purpose

This repo exists to transform design and canon into readable, publishable products.

It includes:

- core book manuscripts
- light novels
- supplements
- publication metadata
- editorial assets
- export-ready files

This repository is not the source of truth for foundational canon or mechanical design.
It is the presentation and publication layer.

## Upstream Source

The source of truth for the universe and system is:

- `transcendence-design`

Publications should remain aligned with the design repository, while applying editorial judgment and reader-facing structure.

## Main Areas

### `core-books/`
Main TTRPG rulebook manuscript and export pipeline.

### `novels/`
Light novels and narrative projects used to reveal the world, species, cultures, and themes.

### `supplements/`
Smaller publication units such as species guides, culture guides, and world expansions.

### `templates/`
Standard templates for chapters, scenes, and supplements.

### `assets/`
Branding and publication support assets.

## Editorial Principles

- clarity before density
- tone consistency
- canon fidelity
- strong terminology discipline
- reader-first structure
- controlled revelation of lore

## Narrative Policy

Novels and narrative works may reveal canon selectively, but should not contradict established truth.

Use `series-bible/` and continuity files to keep narrative cohesion across volumes.

## Core Book Policy

The core book should present rules in a clean and learnable order.
Do not assume readers have access to internal design documents.

## Workflow

1. confirm the design source
2. adapt content to reader-facing form
3. review terminology and tone
4. run editorial QA
5. export publication-ready assets
6. track edition/version differences

## AI Usage Notes

AI assistants in this repo should prioritize:

- readability
- consistency of tone
- continuity
- editorial polish
- structured outputs

They should not create new canon casually.
Any major canon invention should be reviewed against `transcendence-design`.

## Long-Term Goal

This repository should become the home of all polished outward-facing Transcendence publications, from the main game book to novels and supplements.
