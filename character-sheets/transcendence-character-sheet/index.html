Alteraciones

★ la severidad no cambia el efecto — solo la dificultad de aplicación y recuperación  ·  RNG = bonificador de rango de la fuente

Alteración L   M   G Leve +Moderado +Grave
Asfixiado
T.E. Tolerancia al inicio de activ.; −1 acum. a todas las tiradas Prep.→0; T.E. para correr / gritar / esfuerzo sostenido fallo → Incapacitado esa activ.
Aterrorizado
−RNG a T.A./T.D./T.C./T.E. que se acercan o se oponen a la fuente T.E. Contención para acercarse / comprometerse con la fuente T.E. Contención para cualquier acción
Atrapado
mov.→0; −RNG a T.A./T.I./T.D./T.E. Agilidad T.E. Agarre para acciones con Ritmo >4 T.E. Agarre para cualquier acción
Aturdido
−RNG T.R./T.C.; próxima activ. perdida Prep.→0; T.E. Tolerancia para no perder activ. siguiente no Reacciones sin T.E. Tolerancia
Cegado ★
−5 a T.A./T.D./T.C./T.E.
Confundido
−RNG T.A./T.I./T.E. que discriminan objetivo o leen intención T.E. Enfoque para elegir objetivo específico en escena ambigua fallo → objetivo más próximo (puede ser aliado)
Congelado
−RNG T.C./T.E. Agilidad; mov.→½ T.E. Aclimatación para correr / técnicas de movimiento T.E. Aclimatación al inicio de activ.; fallo → pierde activ.
Conmocionado
−RNG T.C./T.E. Compostura e Intelecto Prep.→0 T.E. Contención para usar T.C./T.E. Comp./Intel.
Corroído
−RNG a T.D. y Bloqueo de zona afectada T.E. Tolerancia para usar zona en bloqueos / intercepciones T.E. Tolerancia para cualquier uso def. de la zona; fallo = zona no protege ese intento
Derribado ★
−3 a todas las tiradas; activ. completa para levantarse
Desarmado
ítem cae; acción de movimiento para recuperarlo ítem cae; acción de movimiento para recuperarlo + agarre de otros ítems inestable
Desequilibrado
−RNG T.D./T.E. Agilidad T.E. Equilibrio para correr / Reacciones / T.E. Fuerza/Agi fallo → Derribado Leve
Desorientado
−RNG T.E. Intelecto/Compostura Prep.→0 T.E. Orientación para usar T.E. Intel./Comp.
Electrizado
ataque / mov. / técnicas de ataque/mov. +RNG Ritmo; T.E. Agi −RNG T.E. Tolerancia para usar Reacciones T.E. Tolerancia para cualquier acción
Ensordecido ★
sin T.C./T.E. auditivos; sin alertas sonoras para amenazas no vistas
Impedido
−RNG T.A./T.I. arma y T.E. Agilidad T.E. Enfoque para atacar / interceptar / recargar no Técnicas de arma
Lacerado
ataque / mov. / T.E. Fuerza/Agi +RNG Ritmo T.E. Tolerancia para atacar / mover T.E. Tolerancia para cualquier acción
Paralizado ★
sin acciones
Sobrecargado
−RNG T.R. T.E. Contención para T.E. Aura/Compostura voluntarias T.E. Contención para cualquier T.E.
